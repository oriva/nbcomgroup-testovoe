# Тестовое задание в nbcomgroup

Необходимо сделать конфигуратор MacBook по параметрам.
Референс: https://www.apple.com/ru/shop/buy-mac/macbook-pro/

Вёрстку и стилизацию необходимо сделать свою, т.е не использовать типовое решение на Bootstrap и тд. в рамках задания нам необходимо оценить в том числе, ваш навык вёрстки
Вызов формы, сама форма, работаем только с самим конфигуратором.

Данные для конфигуратора: https://www.iport.ru/upload/front/data.json (признак комплекта - "bundle": true ; признак неактивного элемента - "add": false)
Выбор каждого параметра необходимо реализовать на радиобаттонах. Выбор каждого параметра должен происходить последовательно. В результате нужно вывести блок с выбранными параметрами конфигурации, отдельно вывести комплектацию. 
Единственное ограничение по технологиям - нативный js, использовать gulp или webpack (с обязательной возможностью собрать проект и запустить виртуальный веб-сервер). 


# Gulp-задачи

Префикс gulp (например: gulp build)

* `default` — основная задача, запускает `build`, `watch` и `serve`.
* `build` — сборка всех файлов, запускает задачи `copy`, `images`, `sprites:png`, `sprites:svg`, `pug`, `scss`, `js`.
* `watch` — запускает слежение за файлами, так что при изменении они автоматически пересобираются.
* `serve` — запускает сервер Browsersync.
* `lint` — последовательно запускает линтеры `lint:js`, `lint:pug`, `lint:scss`.
* `lint:js` — проверяет JavaScript-файлы линтером [ESLint](http://eslint.org/).
* `lint:pug` — проверяет Pug-файлы линтером [pug-lint](https://github.com/pugjs/pug-lint).
* `lint:scss` — проверяет SCSS-файлы линтером [stylelint](https://stylelint.io/).
* `optimize:svg` — оптимизирует и форматирует код SVG-файлов в папке `src/images`.
* `optimize:images` — оптимизирует изображения в папке `src/images`.
* `zip` — создает архив проекта.
